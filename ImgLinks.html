Uso de etiquetas multimedias y enlaces en HTML

Etiqueta <img>

Se usa para insertar imagenes. Es una etiqueta vacia, no tiene cierre

Etiqueta <video>

Permite insertar videos. Se recomienda usar varios formatos para compatibilidad

Video de Youtube

Para insertar un video de Youtube, usa la etiqueta <iframe>.

Feed de instagram

Para insertar un feed de Instagram, usa la etiqueta <blockquote>.

Etiqueta <a>

Sirve para crear enlaces a otras paginas, documentos o ubicaciones.
